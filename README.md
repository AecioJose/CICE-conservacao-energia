# 🌱 Energia Consciente: Pequenas Atitudes, Grandes Impactos

Projeto educacional interativo desenvolvido para promover a **conscientização sobre o uso eficiente do ar-condicionado e a conservação de energia** em ambientes institucionais e residenciais.

---

## 📌 Sobre o Projeto

Esta aplicação web apresenta uma **cartilha digital interativa**, com conteúdo educativo, visual moderno e um quiz dinâmico, com o objetivo de:

* Reduzir o desperdício de energia
* Promover hábitos sustentáveis
* Educar sobre impactos ambientais e econômicos
* Incentivar a criação de uma cultura de responsabilidade coletiva

O projeto foi desenvolvido como parte de uma iniciativa da **CICE (Comissão Interna de Conservação de Energia)**.

---

## 🎯 Objetivos

* Conscientizar sobre o uso correto do ar-condicionado
* Demonstrar impactos ambientais (emissão de carbono, consumo energético)
* Incentivar boas práticas no dia a dia
* Ensinar como escolher equipamentos eficientes
* Explicar o dimensionamento correto de ar-condicionado (BTUs)
* Engajar usuários através de um quiz interativo

---

## 🚀 Funcionalidades

### 📖 Cartilha Educativa

* Introdução ao consumo consciente
* Impactos ambientais, econômicos e sustentáveis
* Boas práticas no uso do ar-condicionado
* Mudança de cultura e responsabilidade coletiva

### 🏠 Dicas para Casa

* Redução de consumo em eletrodomésticos
* Uso eficiente de iluminação
* Ventilação natural
* Economia com chuveiro, geladeira e outros equipamentos

### ⚡ Eficiência Energética

* Explicação sobre selos:

  * PBE (Inmetro)
  * Procel
  * CONPET
* Como escolher equipamentos mais eficientes

### ❄️ Dimensionamento de Ar-Condicionado

* Explicação sobre BTUs
* Regra prática de cálculo
* Exemplo real aplicado
* Link para calculadora online

### 🧠 Quiz Interativo

* 5 perguntas de múltipla escolha
* Feedback imediato (acerto/erro)
* Destaque visual das respostas
* Animações e confete ao acertar
* Reinício do quiz

## 🌍 Impacto e Relevância

O projeto aborda um problema real:

> O uso ineficiente de ar-condicionado é uma das principais fontes de desperdício energético em ambientes fechados.

### Por que isso importa?

* 🔥 Aumenta emissões de carbono
* 💸 Eleva custos financeiros
* 🌎 Agrava mudanças climáticas

A proposta é simples:

> **Pequenas ações individuais → Grande impacto coletivo**

---

## 🧠 Decisões Técnicas (Análise Crítica)

* Uso de **HTML único**: facilita distribuição, mas limita escalabilidade
* Uso de **CDN (Tailwind, AOS)**: rápido para prototipagem, mas depende de internet
* **JS puro**: reduz complexidade, mas pode dificultar manutenção em projetos maiores

Se o projeto crescer, melhorias possíveis:

* Separar CSS e JS em arquivos próprios
* Migrar para framework (React, Next.js)
* Adicionar backend para métricas reais de uso
* Implementar PWA (funcionar offline)

---

## 📊 Nível de Confiança do Conteúdo

* Conceitos de eficiência energética: **Alto (baseado em diretrizes reais como Procel/Inmetro)**
* Cálculo de BTUs: **Moderado-Alto (regra prática, não substitui cálculo técnico profissional)**
* Impacto ambiental: **Alto (consenso científico)**
